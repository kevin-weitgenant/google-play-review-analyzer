## 1) Backend — New Endpoint

### Schemas (`app/schemas/api.py`)

Added two new Pydantic models:

```python
class GenerateReplyRequest(BaseModel):
    user_name: str       # Name of the reviewer
    review_content: str  # The review text

class GenerateReplyResponse(BaseModel):
    reply: str           # Generated reply text
```

### Route (`app/routers/reviews.py`)

Added `POST /api/reviews/generate-reply`:

- Receives `GenerateReplyRequest`
- Logs `user_name` and `review` to the backend console (`logger.info`)
- Returns `GenerateReplyResponse` with a placeholder reply

## 3) Frontend — Wiring It Up (`design-kanban.tsx`)

### Replaced fake function with real API call

```tsx
// BEFORE — simulated, client-side only
function simulateAIResponse(review, _config): Promise<string> {
  // setTimeout + hardcoded string
}

// AFTER — real backend call
async function generateReply(review: Review): Promise<string> {
  const request: GenerateReplyRequest = {
    user_name: review.user_name,
    review_content: review.content,
  }
  const api = getReviews()
  const response = await api.generateReplyApiReviewsGenerateReplyPost(request)
  return response.reply
}
```

### Updated `handleGenerateResponse`

Changed one line — from `simulateAIResponse(review, config)` to `generateReply(review)`.

### Added imports

```tsx
import { getReviews } from "../api/reviews/reviews"
import type { GenerateReplyRequest } from "../api/models"
```
